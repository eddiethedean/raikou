# raikou documentation

**raikou** is the umbrella package for Spark helpers in Python; **raikou-core**
provides the low-level Spark execution engine.

| Document | Description |
|----------|-------------|
| [Getting started](getting_started.md) | Install, connect, create a frame, collect |
| [Development](development.md) | Monorepo workflow (links to contributing guide) |
| [Roadmap](roadmap.md) | Planned directions |

## Packages

| PyPI | Role |
|------|------|
| `raikou` | `connect`, `RaikouDataFrame`, `col` / `lit` |
| `raikou-core` | `SparkExecutionEngine`, plan + root helpers |
